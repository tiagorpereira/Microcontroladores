1. O que são sistemas embarcados?

	Os sistemas embarcados são sistemas computacionais destinados a funcionalidades específicas e limitadas. Diferentemente dos sistemas gerais estes 
        são projetados para atender um conjunto específico de requisitos e realizam determinadas funções condizentes com tais requisitos.

2. O que são sistemas microprocessados?

	Os sistemas microprocessados são sistemas eletrônicos constituídos de microcontroladores. Estes são desenvolvidos com um código próprio de funcionamento,
	ou seja, não rodam um sistema operacional mas um programa com funções específicas.
